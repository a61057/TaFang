# ⚔ 塔防 - Tower Defense

一款基于 Canvas 的塔防游戏，支持战役/无尽两种模式，搭载英雄系统、阵营羁绊、随机事件、天气昼夜等丰富机制。

## 特性

- **双模式**：50波战役关卡 + 无尽模式
- **英雄系统**：4种英雄（侦察兵/重装兵/突击兵/狙击手），可招募切换，保留等级经验
- **武器商店**：6种武器可购买装备，每位英雄最多装备3件
- **5种炮塔**：加农炮/机枪/迫击炮/冰霜塔/特斯拉塔，可升至3级
- **阵营羁绊**：人类/精灵/机械/亡灵，3/6/9层共鸣加成
- **随机事件**：20种事件，每波35%概率触发
- **天气昼夜**：5种天气 + 昼夜循环影响战场
- **成就系统**：16个成就解锁
- **存档读档**：3个存档位
- **键盘操控**：WASD移动坦克，方向键控制光标，空格交互

## 技术栈

- **Canvas 2D** 纯前端渲染（960×720，24×18网格）
- **ES Modules** 组织代码
- **Electron** 桌面打包（v28.3.3）
- **对象池** 管理敌人/子弹/粒子

## 快速开始

```bash
npm install
npm run serve    # 开发服务器 → http://localhost:3000
npm start        # Electron 桌面运行
npm run pack     # 打包为 Windows 应用
```

## 目录结构

```
src/
├── js/
│   ├── config/      # 配置、数据定义、多语言
│   ├── engine/      # 游戏循环、粒子、对象池
│   ├── entities/    # 英雄、敌人、炮塔、子弹
│   ├── managers/    # 波次/阵营/事件/天气/英雄管理
│   ├── ui/          # HUD、菜单、面板
│   ├── map/         # 地图
│   ├── audio/       # 音频
│   └── save/        # 存档
├── css/
└── html/
```
